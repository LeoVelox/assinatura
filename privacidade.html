SarmTech Gestão Inteligente

Política de Privacidade

Conforme a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018)

Última atualização: Janeiro de 2024

Transparência é nossa prioridade: Esta política explica como coletamos, usamos e protegemos seus dados no sistema SarmTech.

1. Quem somos

SarmTech é uma plataforma de gestão empresarial que oferece o sistema OS_flex V3 para empresas de serviços técnicos. Atuamos como Controlador dos dados pessoais conforme definido pela LGPD.

2. Dados que Coletamos

Coletamos diferentes tipos de dados para fornecer nossos serviços:

2.1 Dados do Usuário (Titular)

Dado Finalidade Base Legal
Nome completo Identificação e comunicação Execução de contrato
E-mail Conta, notificações e suporte Execução de contrato
CPF/CNPJ Identificação fiscal e segurança Obrigação legal
Telefone Suporte e comunicação Execução de contrato
Endereço da empresa Emissão de documentos Execução de contrato

2.2 Dados de Clientes (Inseridos por você)

Quando você cadastra clientes em nosso sistema, coletamos:

  • Nome do cliente/empresa
  • Documento (CPF/CNPJ)
  • Contato (telefone, e-mail)
  • Endereço (para serviços)
  • Histórico de serviços realizados

Importante: Você é o Controlador desses dados e deve obter consentimento dos seus clientes conforme LGPD.

3. Como Usamos Seus Dados

Seus dados são usados exclusivamente para:

  • Fornecer e melhorar nosso sistema
  • Processar pagamentos e gerenciar sua conta
  • Enviar notificações importantes do sistema
  • Oferecer suporte técnico
  • Cumprir obrigações legais
  • Prevenir fraudes e aumentar segurança

Não vendemos seus dados para terceiros. Não usamos seus dados para marketing de terceiros.

4. Compartilhamento de Dados

Compartilhamos dados apenas quando necessário:

Com quem O que compartilhamos Por quê
Processadores de pagamento Dados necessários para cobrança Processamento de pagamentos
Supabase (hospedagem) Todos os dados do sistema Hospedagem segura do banco de dados
Autoridades legais Dados solicitados judicialmente Cumprimento de obrigação legal

Supabase: Usamos o Supabase para hospedar nossos dados. Eles possuem certificações de segurança e estão em conformidade com regulamentações de proteção de dados.

5. Segurança dos Dados

Implementamos medidas de segurança robustas:

  • Criptografia: Dados em trânsito (HTTPS/TLS) e em repouso
  • Row Level Security: Cada usuário só acessa seus próprios dados
  • Backups automáticos: Diários com retenção de 30 dias
  • Autenticação segura: JWT com expiração controlada
  • Proteção contra ataques: Firewall e monitoramento 24/7

6. Retenção de Dados

Mantemos seus dados enquanto você for usuário ativo. Após cancelamento:

  • 30 dias: Dados mantidos para possível restauração
  • Após 30 dias: Exclusão definitiva dos dados
  • Dados fiscais: Mantidos por 5 anos (obrigação legal)

7. Seus Direitos na LGPD

Conforme a Lei Geral de Proteção de Dados, você tem direito a:

  • Confirmação: Saber se tratamos seus dados
  • Acesso: Solicitar cópia de seus dados
  • Correção: Atualizar dados incompletos ou errados
  • Exclusão: Pedir exclusão de dados desnecessários
  • Portabilidade: Receber dados em formato estruturado
  • Revogação: Retirar consentimento a qualquer momento

9. Cookies e Tecnologias Similares

Usamos cookies essenciais para:

  • Manter sua sessão ativa
  • Lembrar suas preferências
  • Melhorar segurança

Não usamos cookies de marketing ou de terceiros.

Podemos atualizar esta política para refletir mudanças legais ou no sistema. Notificaremos alterações significativas por e-mail com 30 dias de antecedência.
