O que é a LGPD?

A Lei Geral de Proteção de Dados (Lei 13.709/2018) é a legislação brasileira que regula o tratamento de dados pessoais. Ela estabelece direitos para os cidadãos e obrigações para as empresas que coletam dados.

Seus Direitos na LGPD

Confirmação

Direito de saber se tratamos seus dados pessoais.

Acesso

Solicitar uma cópia dos dados que temos sobre você.

Correção

Atualizar dados incompletos, inexatos ou desatualizados.

Exclusão

Solicitar a eliminação de dados desnecessários ou excessivos.

Portabilidade

Receber dados em formato compatível para transferência.

Revogação

Retirar consentimento dado para tratamento de dados.

Como Exercer Seus Direitos

  1. Envie um e-mail para lgpd@sarmtech.com
  2. Identifique-se (nome completo e e-mail cadastrado)
  3. Especifique qual direito deseja exercer
  4. Responderemos em até 15 dias úteis

Para usuários do sistema: Lembre-se que você também é responsável pelos dados de seus clientes que insere no sistema. Certifique-se de ter consentimento adequado para tratar esses dados.

Dúvidas Frequentes

Vocês vendem meus dados?

Não. Não vendemos, alugamos ou compartilhamos seus dados com terceiros para marketing.

Como meus dados são protegidos?

Usamos criptografia, controle de acesso rigoroso e hospedagem segura no Supabase com certificações internacionais.

Posso excluir minha conta e todos meus dados?

Sim. Basta solicitar via e-mail. Excluiremos todos seus dados pessoais, exceto informações necessárias para obrigações legais.
